# Token Based Authentication

Learning token-based authentication using JWT, Node.js (Express) and vanilla JS.

## What it covers
- User login with JWT generation (`jwt.sign`)
- Protected route with JWT verification (`jwt.verify`)
- Sending JWT via `Authorization: Bearer` header from the frontend

## Tech Stack
- Node.js + Express
- jsonwebtoken
- Vanilla HTML + JS

## Setup

1. Clone the repo
   git clone [https://github.com/Prasad1-S/token-based-auth.git](https://github.com/Prasad1-S/token-based-auth.git)
   cd token-based-auth

2. Install dependencies
   npm install

3. Create a .env file in the root
   SECRET=your_secret_key_here

4. Start the server
   node server.js

5. Open index.html directly in your browser
